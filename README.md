# Heart Sound Segmentation

This project implements a heart sound segmentation pipeline using Temporal Convolutional Networks (TCNs) with Bidirectional GRU and a small Flask web demo.

Demo video: https://drive.google.com/file/d/1tvySmdtZuvQ9UUrqrPvRbymJEYDkfvyO/view?usp=sharing

- This was developed as a term project for BME 310 "Communicating Protocols for Biomedical Instruments Sessional".
- Training is designed to work with the Circor Digiscope Phonocardiogram dataset 1.0.3.
